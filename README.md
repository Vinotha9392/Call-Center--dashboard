## Call Center Dashboard using Power BI

## Project Overview
Developed an interactive Call Center Dashboard in Power BI to analyze customer service performance, call handling efficiency, customer sentiment, and operational trends.
The dashboard provides meaningful insights into call volumes, response time, customer issues, service channels, and agent performance through dynamic visualizations.

## Project Objective
The objective of this project is to analyze call center operational data and transform raw datasets into an interactive dashboard that helps monitor customer interactions, service performance, and operational efficiency for better business decision making.

## Tools Used
- Power BI
- Excel / CSV
- DAX
- Power Query

## Dashboard Pages

# 1. Home Dashboard
Overview of total call center performance metrics
Total calls, call duration (minutes & hours), average call duration, and response time summary
Interactive dashboard for monitoring overall service performance

# 2. Call Performance Analysis
- Day-wise total calls analysis
- State-wise call distribution using map visualization
- Channel-wise call analysis (Call Center, Chatbot, Email, Web)
- Customer sentiment analysis (Positive, Neutral, Negative, Very Negative)
- Response time analysis based on SLA performance
- Call reason analysis (Billing, Payments, Service Issues)

# 3. Customer Details Dashboard
- Detailed customer interaction records
- Customer name, city, communication channel, call reason, sentiment analysis
- Call duration tracking for customer service monitoring
- Filter-based customer data analysis with date, state, city, and CSAT score
